Пак с черно-белыми 32битными текстурами для Quake2.
Автор: Tosher
WWW: http://tosher.h16.ru
----------------------------------------------------------------

Основан на моей сборке с цветными 32-битными текстурами для Quake2.
Пак включает в себя все необходимые текстуры для карт:
q2dm1, q2dm2, q2dm3, q2dm4, q2dm5, q2dm6, q2dm7, q2dm8, 
match1, ztn2dm1, ztn2dm2, ztn2dm3, ztn2dm4, ztn2dm5, fury, ptrip
Частично он будет покрывать и другие карты, но только частично.

Скриншоты доступны через веб:
http://tosher.h16.ru/screens.php

Установка:
================================================================
Для r1q2:
- Требуется PNG или JPG версия данного пака
- Требуется библиотека R1GL (http://www.r1ch.net/stuff/r1gl/)

Удалите все дополнительные паки с текстурами из каталога quake2\baseq2.
Скопируйте данный пак в папку quake2\baseq2.

Для получения корректной черно-белой темы запустите команду:
---
set gl_coloredlightmaps "0"
vid_restart
или просто выполните скрипт:
exec bw_r1q2.cfg 
(включен в данный пак)
---
Для будущего использования рекоммендуется добавить данную команду в ваш quake2 конфиг.
================================================================
Для q2advance:
- Требуется PNG версия данного пака

Удалите все дополнительные паки с текстурами из каталога quake2\baseq2.
Скопируйте данный пак в папку quake2\baseq2.

Для получения корректной черно-белой темы запустите команду:
---
set r_coloredlightmaps "0.01"
или просто выполните скрипт:
exec bw_q2a.cfg 
(включен в данный пак)
---
Для будущего использования рекоммендуется добавить данную команду в ваш quake2 конфиг.
================================================================
Для nocheat:
- Требуется JPG версия данного пака
- Требуется библиотека NCGL

Удалите все дополнительные паки с текстурами из каталога quake2\baseq2.
Скопируйте данный пак в папку quake2\baseq2.
Переименуйте его в pakNN.pak, где NN - максимальный номер из имеющихся паков, от 1 до 99.

Для получения корректной черно-белой темы запустите команду:
---
set gl_coloredlightmaps "0.01"
vid_restart
или просто выполните скрипт:
exec bw_nc.cfg 
(включен в данный пак)
---
Для будущего использования рекоммендуется добавить данную команду в ваш quake2 конфиг.
================================================================
Для aprq2:
- Требуется JPG или PNG версия данного пака

Удалите все дополнительные паки с текстурами из каталога quake2\baseq2.
Скопируйте данный пак в папку quake2\baseq2.
Переименуйте его в pakNN.pak, где NN - максимальный номер из имеющихся паков, от 1 до 99.
В конфиге необходимо включить опцию fs_allpakloading 1, перед запуском Quake2

Для получения корректной черно-белой темы запустите команду:
---
set gl_coloredlightmaps "0"
vid_restart
или просто выполните скрипт:
exec bw_aprq2.cfg 
(включен в данный пак)
---
Для будущего использования рекоммендуется добавить данную команду в ваш quake2 конфиг.
